# 目標設定ガイド

## 1. 月目標設定

1. 管理者ログイン後、`月目標設定` を開く
2. 対象月を選択
3. 部署ごとの数値を入力して保存

## 2. 路程設定（期間）

1. `路程設定` を開く
2. 対象年月、路程回次、開始日、終了日を入力
3. `路程を保存` を押す

## 3. 路程目標入力

1. `対象路程` を選択
2. 部署ごとの目標値を入力
3. `路程目標を保存` を押す

## 4. 上書き確認ダイアログ

以下の場合、保存時に確認ダイアログが表示されます。

- 同じ年月・同じ回次（同名路程）がすでにある
- 別名でも期間が重複している

`OK` を押すと上書き、`キャンセル` で保存中止です。

## 5. 削除

- 作成済み路程目標一覧の `削除` で路程を削除できます。
