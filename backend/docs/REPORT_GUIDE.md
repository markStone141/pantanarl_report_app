# 報告者向けガイド

## 1. ログイン

1. ログイン画面で `REPORT` を選択
2. パスワードを入力してログイン
3. 報告対象の部署ボタンを押して報告フォームへ移動

## 2. 報告の送信

1. 報告日・責任者を選択
2. メンバー行を入力
3. `送信` を押す

## 3. 同日再送信の挙動（重要）

- 同じ日・同じ部署にすでに報告がある場合、新規で増えずに上書きされます。
- その日の同部署データは最終送信内容が1件として保持されます。

## 4. 修正と削除

- 送信履歴から `編集` で内容を修正できます。
- 誤送信時は `削除` を実行します。

## 5. 注意点

- WV は `CS件数` と `難民件数` を分けて入力します。
- Style 部署は現場入力が不要です。
