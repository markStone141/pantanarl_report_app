# 管理者運用ガイド

## 日次の基本フロー

1. 提出状況一覧を確認
2. 本日の部門別実績を確認
3. 目標管理（当月・路程）との進捗差分を確認
4. `メール本文作成` で本文を生成し、プレビュー・コピー

## メール本文作成

- モード切替で当日/前日データを参照できます。
- テンプレートはDBの報告・目標値から自動生成されます。
- 路程見出しは `第〇次路程（日付）` 形式で表示されます。

## よくある運用ルール

- 報告修正は報告画面の `編集` で対応
- 同日同部署の再送信は上書き（重複蓄積しない）
- 目標修正は月目標/路程目標から再保存

## トラブル時

- 表示が古い場合: ブラウザをハードリロード
- 入力反映が不自然な場合: 対象日の履歴と管理画面の値を突合
- 本番障害時: Cloud Run の前リビジョンへ一時ロールバック
